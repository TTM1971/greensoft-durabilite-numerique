# Indicateurs de Performance (KPI) – Durabilité Numérique

## 1. Infrastructures infonuagiques
| Indicateur | Description | Objectif |
|------------|-------------|-----------|
| Taux de virtualisation | Proportion des serveurs virtualisés | > 85 % |
| Utilisation des instances cloud | Ratio des instances actives vs allouées | > 70 % |
| Consommation énergétique | kWh annuels des environnements cloud | -20 % d’ici 2 ans |
| Émissions liées au cloud | tCO₂e annuelles | -25 % d’ici 2030 |

## 2. Écoconception logicielle
| Indicateur | Description | Objectif |
|------------|-------------|-----------|
| Écoscore moyen | Score énergétique des applications | ≥ 75/100 |
| Réduction du volume de données | Nettoyage et archivage | -30 % |
| Empreinte carbone par service | gCO₂e/transaction | -15 % |
| Projets conformes aux normes internes | % des projets écoconçus | 100 % |

## 3. Gestion des équipements
| Indicateur | Description | Objectif |
|------------|-------------|-----------|
| Taux de réutilisation des composants | Pièces réutilisées | ≥ 40 % |
| Déchets électroniques recyclés | kg/an | +30 % |
| Durée de vie des équipements | Moyenne en années | +2 ans |
| Achats certifiés | EPEAT/Energy Star | 100 % |

## 4. Optimisation énergétique interne
| Indicateur | Description | Objectif |
|------------|-------------|-----------|
| Consommation par poste | kWh/an/poste | -20 % |
| Postes éteints après heures | % | > 90 % |
| Taux de télétravail adopté | % | ≥ 40 % |
| Intensité énergétique | kWh/employé | -25 % |
