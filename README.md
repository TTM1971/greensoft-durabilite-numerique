
# Stratégie de Durabilité Numérique – GreenSoft Solutions
### Projet 6 — INF1753 : Pratique professionnelle et communication en informatique  
Université du Québec en Outaouais (UQO) — Automne 2025

---

## Description du projet

Ce dépôt contient la documentation officielle réalisée dans le cadre du Projet 6 – Stratégie de Durabilité Numérique et Informatique Verte.  
Ce projet vise à accompagner GreenSoft Solutions, une entreprise technologique offrant des services infonuagiques et logiciels, dans la réduction de son empreinte environnementale.

Le dépôt regroupe la charte numérique verte, les indicateurs de performance (KPI), les modèles d’audit énergétique ainsi que la documentation de gouvernance liée à la durabilité numérique.

---

## Objectifs pédagogiques

Le projet s’inscrit dans la formation INF1753 et vise à :

- comprendre les enjeux environnementaux du numérique ;
- identifier et appliquer des pratiques Green IT ;
- intégrer les normes ISO 14001, ISO 50001 et l’Analyse du Cycle de Vie (ACV) ;
- relier durabilité, éthique, gouvernance et responsabilité sociale ;
- utiliser Git/GitHub pour assurer la documentation et la traçabilité des projets durables ;
- développer des compétences de communication professionnelle.
